# 「个人数据的围城」工作坊

## 关于本工作坊

这是「开放数据中国」出品的「普通人的数字权利指北」系列工作坊之一：「个人数据的围城」。

### 工作坊简介

在一个越发数字化的社会中，我们早已习惯了身边的数字工具，它们无时无刻不在采集我们的数据：运动手表、点餐小程序、App、监控摄像头等等，而我们也在不断生产着数据：例如写微博、发点评、点赞短视频等等。我们的个人数据正如同一种「社会默契」下的「新货币」，成为我们日常生活中的一环。但我们真的愿意失去对它的掌控吗？

### 工作坊内容

1. **感知个人数据的流动**：从日常场景切入，感知个人数据在不同服务主体间的流动
2. **个人信息、个人数据和隐私**：理解基本概念和差异
3. **采集获取个人数据**：了解为什么和如何采集个人数据，认识 SDK 这个「沉默」的数据捕获者
4. **个人数据流动画布**：通过互动任务，可视化感知中的数据流动
