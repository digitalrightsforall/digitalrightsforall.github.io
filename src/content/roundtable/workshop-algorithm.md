# 「我们身边的算法决策」工作坊

## 关于本工作坊

这是「开放数据中国」出品的「普通人的数字权利指北」系列工作坊之二：「我们身边的算法决策」。

### 工作坊简介

算法并不是魔法，它既不一定高深，也不一定复杂。我们每个人学习过的算式、公式都是一种算法。但进入数字时代，为何我们会谈算法而色变呢？本工作坊将带领大家感知算法决策、认知算法的问题和局限。

### 工作坊内容

1. **感知算法决策**：理解算法是什么，体验生活中的算法应用场景
2. **我们的算法体验**：通过互动投票，了解大家因算法遭受过的困扰
3. **谁掌控决策**：理解自动化决策的概念，探讨人和算法谁应该做主
4. **认知算法决策**：通过设计招聘决策产品和玩游戏，理解算法的问题和局限
