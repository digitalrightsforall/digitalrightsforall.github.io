# 双周刊 第4期：数字权利动态（国庆特刊）

## 假期网络安全提示

### 连接公共WiFi要当心

国庆假期出行，公共WiFi虽方便，但风险也不小：

**风险点：**
- 数据可能被窃取
- 可能遇到钓鱼热点
- 账户密码易泄露

**安全建议：**
1. 尽量使用手机流量
2. 如需使用公共WiFi，避免进行网银、支付等操作
3. 使用VPN加密连接
4. 关闭自动连接WiFi功能

## 购物节隐私保护

### 双十一前先"设防"

购物节即将来临，在买买买之前，先做好隐私保护：

**购物时注意：**
1. 只在官方和可信平台购物
2. 谨慎填写个人信息
3. 仔细阅读隐私政策
4. 开通账户变动通知

**快递单处理：**
- 涂抹或撕毁个人信息后再丢弃
- 可使用专用的隐私保护印章
- 电子面单同样要注意删除

## 旅行数据安全

### 出行前的数据准备

国庆出游，数据安全也不能忘：

**出行前：**
1. 备份重要数据
2. 开启设备的远程查找和擦除功能
3. 更新系统和应用到最新版本
4. 准备备用电源和离线地图

**旅行中：**
1. 设备不离视线
2. 谨慎使用公共充电设备
3. 避免在公共设备上登录账户
4. 拍照时注意背景敏感信息

## 社交媒体晒照提醒

### 晒图也要"脱敏"

假期拍照发朋友圈是标配，但有些信息不能晒：

**谨慎晒出：**
- 火车票、机票（含个人信息）
- 家门钥匙或车牌
- 孩子的学校和班级信息
- 具体的行程安排

**建议做法：**
- 对敏感信息进行打码处理
- 延迟发布或分组可见
- 关闭位置信息显示

## 紧急情况处理

### 数据丢失或泄露怎么办？

如果不幸遇到数据问题：

1. **设备丢失**：立即远程锁定和擦除数据
2. **账户被盗**：尽快修改密码，联系平台客服
3. **信息泄露**：关注相关账户动态，必要时冻结
4. **寻求帮助**：可向监管部门或专业机构求助
